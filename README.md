# 🌏 신의 손 — 문명의 하늘

하늘에서 도시를 내려다보며 문명을 지켜보고, 재해를 내리는 **3인칭 전지(God) 시뮬레이션 게임**.
Three.js 단일 HTML 파일로 만들어져 브라우저에서 바로 실행됩니다.

## 🎮 플레이 방법

브라우저에서 `index.html`을 열면 바로 시작됩니다. (인터넷 연결 필요 — three.js CDN 사용)

| 조작 | 동작 |
|---|---|
| 🖱️ 휠 | 확대 / 축소 |
| 🖱️ 휠 누르고 드래그 | 카메라 각도 회전 |
| 🖱️ 왼쪽 드래그 / WASD | 이동 |
| 🖱️ 사람 클릭 | 이름·직업·가족·상태 보기 |

## ✨ 주요 기능

- **문명 진화 9시대** — 공룡시대 → 원시 → 농경 → 중세 → 1·2·3·4차 산업혁명 → 우주시대. 시대마다 건물·직업·탈것이 바뀌고, 운석 충돌 등 큰 사건이 연출됩니다.
- **시간 조작** — 배속 슬라이더로 빨리 감기, 일시정지, **거꾸로 되돌리기**(⏪)까지. 문명 타임라인 슬라이더로 원하는 시대로 즉시 점프.
- **재해 3종** — 💨 바람(나무가 휘고 사람들이 밀림), 🌋 지진(건물 붕괴·카메라 흔들림), 🌊 홍수(강 범람).
- **살아있는 사람들** — 각자 이름·직업·가족이 있고, 아침에 출근하고 밤에 잠들며, 마주치면 말풍선으로 인사하고 소리를 냅니다.
- **낮밤 & 사계절** — 새벽·한낮·노을·밤이 흐르는 물리 기반 하늘(Sky 셰이더), 봄 벚꽃·여름 녹음·가을 단풍·겨울 눈.
- **크기 변신** — 🛰️ 하늘의 신 → 🗿 거인 → 🚶 사람 → 🐜 곤충 시점까지 자유롭게.
- **우주 스케일 줌** — 끝까지 축소하면 둥근 지구 → 태양계(행성 8개 + 달 공전 + 궤도선) → 🌌 우리은하까지. 행성을 클릭해 포커스하고, 지구·달을 확대하면 착륙합니다.
- **달나라 여행** — 우주시대에 🚀 로켓을 발사하면 대기권을 뚫고 우주에서 달까지 실제로 비행한 뒤 착륙, 우주인으로 걸어다닐 수 있습니다 (낮은 중력 껑충껑충).
- **✋ 파괴 모드** — 신의 손으로 건물을 클릭해 부수고(잔해가 튐), 사람을 하늘로 던질 수 있습니다. 부서진 건물은 시간이 지나면 복구됩니다.
- **그래픽** — 실시간 반사 강물(Water), 창문이 켜지는 건물 외벽 텍스처, 팔다리 달린 캐릭터 걷기 애니메이션, Bloom 후처리, ACES 톤매핑, 부드러운 그림자.
